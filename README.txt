CÓMO SUBIR A GITHUB:
1. Entrá al repositorio.
2. Subí estos archivos en la raíz del proyecto.
3. Activá GitHub Pages desde Settings > Pages.
4. Abrí la URL pública desde Chrome en el celular.
5. Tocá los tres puntos > Agregar a pantalla principal.

IMPORTANTE:
La app no inventa precios reales. Para recomendaciones operativas, cargá datos reales del día en formato JSON.
El botón "Cargar ejemplo educativo" es solo para probar la vista y el motor de análisis.
